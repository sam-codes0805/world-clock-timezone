# World Clock (Timezone Based)

A web application that displays real-time time based on selected timezone using API.

## Features
- Select different timezones
- Displays accurate real-time time
- Dynamic updates using API
- Clean and simple UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Timezone API
